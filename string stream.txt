String Stream in C++

Header File : #include<sstream>

Methods -
1) sso.str()   : Returns string from stream
2) sso.str("") : Empty the contents of stringstream. (Replace old content by new content of stringstream)
3) sso.clear() : Empty the bit flags.
4) sso.eof()   : It returns true when stringstream is ended.
Note :- A non-empty stringstream object has bit flags indicating its status as good, bad, eof, or fail.
Note :- If it is not possible to convert string to int, then stringstream extracts 0 as result.
Note :- stringstream does not read whitespace as input.
Note :- stringstream divides space-separated data into separate strings (On the basis on whitespace). 
Note :- stringstream extracts one space-separated string at a time.

In order to use String Stream, we have to create an object of it first.
Like other streams, here we can also use insertion and extraction operators.

Extract Integer from stringstream (Important) :
Concept : 
Concept : We should not directly extract integer from stringstream, 
#include <iostream>
#include <sstream>
using namespace std;

void extractIntegerWords(string str)
{
   stringstream sso;

   /* Storing the whole string into string stream */
   sso << str;

   /* Running loop till the end of the stream */
   string temp;
   int found;
   while (!sso.eof()) {

      /* extracting word by word from stream */
      sso >> temp;

      /* Checking the given word is integer or not. If word is not integer, then found becomes 0. */
      if (stringstream(temp) >> found)
         cout << found << " ";

      /* To save from space at the end of string */
      temp = "";
   }
}

// Driver code
int main()
{ 
   string str = "1: 2 3 4 prakhar";
   extractIntegerWords(str);

   return 0;
}
